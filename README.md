## Harry Potter React App  
### A React + Vite app that calls a Harry Potter API and displays the data. Work in progress.  

## to Run:  
### clone this repo to your local machine  
### open a terminal in the project's root directory  
### type ```npm install```  
### type ```npm run dev```
